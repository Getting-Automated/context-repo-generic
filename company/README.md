# Company Context

Use this folder for stable context about the organization, product, service, market, and communication style.

Recommended files:

- [positioning.md](positioning.md): what the business or project is, who it serves, and how it should be described
- [offers.md](offers.md): products, services, packages, pricing notes, or engagement models
- [voice-and-style.md](voice-and-style.md): writing style, tone, phrases to use, and phrases to avoid

Keep this folder focused on durable context. Do not paste every website page, sales deck, or transcript here.
