# Decision Records

Use this folder to record decisions that agents should not relitigate every time.

Decision records are useful for:

- architecture choices
- positioning choices
- policy decisions
- workflow rules
- tool selections
- scope boundaries

Copy [decision-record.template.md](decision-record.template.md) when adding a new decision.

Suggested filename:

```text
YYYY-MM-DD-short-decision-name.md
```
