# Projects

Use this folder for project-specific context.

Project briefs should point to the stable context they rely on instead of copying the whole repo into each project.

Copy [project-context.template.md](project-context.template.md) for new projects.

Suggested structure:

```text
projects/
|- README.md
|- project-context.template.md
`- example-project/
   `- brief.md
```
