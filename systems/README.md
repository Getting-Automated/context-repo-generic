# Systems And Sources Of Truth

Use this folder to tell agents where facts live and what systems are safe to use.

Recommended files:

- [source-of-truth.md](source-of-truth.md): which systems own which facts
- [access-and-sensitivity.md](access-and-sensitivity.md): what data is sensitive and what tools may access it

This folder should map systems. It should not copy every record from those systems.
