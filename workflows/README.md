# Workflows

Use this folder to describe repeatable work.

A good workflow file should tell an agent:

- what starts the process
- required inputs
- expected outputs
- deterministic rules
- fuzzy judgment steps
- owners
- failure paths
- review requirements

Start with [default-workflow.md](default-workflow.md), then copy [../templates/scoped-task.template.md](../templates/scoped-task.template.md) for specific workflows.
