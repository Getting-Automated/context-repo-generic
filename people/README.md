# People And Ownership

Use this folder to help agents understand who owns decisions, systems, workflows, and reviews.

Recommended files:

- [roles-and-ownership.md](roles-and-ownership.md): team roles and responsibility boundaries
- [stakeholders.md](stakeholders.md): internal or external stakeholders an agent should know about

Do not store sensitive HR data, compensation, private performance notes, or personal information that does not belong in agent context.
